# 健身打卡APP — 情侣版 · 开发提示词

## 1. 项目概述

**名称：** FitTogether（暂定，可改）

**定位：** 一款面向情侣的安卓健身打卡应用。两人可以组队打卡，互相监督、互动，让健身变得更有趣和有动力。

**平台：** Android（Kotlin + Jetpack Compose）

**风格：** 年轻化、活力感、毛玻璃效果、流畅微交互、渐变色彩

---

## 2. 核心功能模块

### 2.1 用户系统
- 手机号注册/登录
- 情侣绑定：通过邀请码配对，双方确认后绑定
- 个人资料：头像（支持自定义上传）、昵称、签名
- 健身目标设定：每周打卡次数目标

### 2.2 打卡模块

**预设打卡类别：**
| 类别 | 图标 | 默认颜色 |
|------|------|----------|
| 🏃 跑步 | 跑步鞋 | 活力橙 #FF6B35 |
| 💪 练胸 | 胸肌轮廓 | 力量红 #E53935 |
| 🏋️ 练背 | 背部线条 | 沉稳蓝 #1E88E5 |
| 🦵 练腿 | 腿部轮廓 | 深紫 #7B1FA2 |
| 🎯 练肩 | 肩部线条 | 青绿 #00897B |
| 🧘 拉伸 | 瑜伽剪影 | 柔和绿 #43A047 |
| 🔥 HIIT | 火焰 | 激情粉 #D81B60 |

**自定义打卡：**
- 用户可创建自定义类别
- 支持类别名、图标（emoji/本地图标库）、颜色
- 支持自定义备注模板（如 "今天跑了X公里"）
- 自定义字段：数值型（公里/分钟/组数）、开关型（完成/未完成）

### 2.3 打卡详情
- 点击类别进入打卡页
- 记录：数值（时长/距离/组数/重量）+ 备注
- 可拍照上传（每组动作照片/跑步路线截图等）
- 心情/感受标签（😊 轻松 😅 一般 🥵 累爆了）
- 完成后弹出打卡成功动画

### 2.4 日历视图
- 月份热力图（类似 GitHub 贡献图）
- 点击日期查看当天打卡详情
- 不同类别用不同颜色标识
- 情侣共同日历：可以看到对方的打卡情况

## 3. 情侣互动功能（重点）

### 3.1 情侣主页
- 双人头像展示 + 已绑定天数
- 共同健身进度条（本周 vs 目标）
- 实时打卡动态（"Ta 刚完成了 5km 跑步！"）

### 3.2 互动提醒
- 轻戳对方（Poke）：发送一个动画提醒对方健身
- 为对方加油：一键发送鼓励短语 + 全屏动画（❤️🔥💪）
- 对方完成打卡后收到推送通知


### 3.5 定制化照片
- 打卡时可拍照或从相册选择
- 情侣共享相册：双方打卡照片汇入同一空间
- 照片支持贴纸/滤镜（健身主题贴纸）
- 可设为打卡背景图

---

## 4. 交互特效（重点）

### 4.1 微交互动画
- **打卡成功：** 全屏撒花/彩带动画 + 触觉反馈
- **连续打卡天数：** 火焰图标从小到大弹跳，数字递增滚动动画
- **点击按钮：** 水波纹扩散效果（Material 3 风格）
- **页面切换：** 共享元素过渡动画
- **下拉刷新：** 自定义健身小人动画

### 4.2 情侣专属特效
- **对方上线：** 爱心气泡飘过屏幕
- **收到加油：** 屏幕震动 + 爱心粒子爆炸效果
- **同时打卡：** 画面出现双人击掌动画
- **挑战完成：** 烟花全屏动画 + 音效

### 4.3 UI 风格
- 毛玻璃（Glassmorphism）导航栏和卡片
- 渐变色背景（根据时间变化：早上暖橙→傍晚紫蓝）
- Lottie 动画图标
- 流畅的列表滚动阻尼效果
- Dark Mode / Light Mode 自动切换

## 7. 技术栈建议

| 层级 | 技术选型 |
|------|----------|
| 语言 | Kotlin |
| UI | Jetpack Compose + Material 3 |
| 导航 | Compose Navigation |
| 本地存储 | Room Database |
| 网络 | Retrofit + OkHttp |
| 图片加载 | Coil |
| 动画 | Lottie + Compose Animation API |
| 推送 | Firebase Cloud Messaging |
| 后端 | Firebase (Firestore + Auth + Storage) |
| 架构 | MVVM + Clean Architecture |

## 8. 页面结构

```
├── 首页（Dashboard）
│   ├── 今日打卡概览卡片

│   ├── 类别快捷打卡按钮（可滚动网格）

├── 打卡详情页
│   ├── 数值输入
│   ├── 照片上传
│   ├── 心情标签
│   └── 打卡按钮（大按钮 + 动画）
├── 日历页
│   ├── 月视图热力图
│   └── 日期详情底部弹窗
├── 情侣空间页
│   ├── 相册
│   ├── 互动记录
│   └── 挑战列表
├── 排行/挑战页
├── 个人中心页
│   ├── 头像/资料编辑
│   ├── 成就墙
│   ├── 自定义类别管理
│   └── 设置（提醒时间、主题等）
```

## 9. 数据模型（核心）

```
User
  - uid, phone, nickname, avatar
  - partnerId (情侣绑定)
  - bindTime, weeklyGoal

CheckInCategory
  - id, name, icon, color
  - isCustom (是否自定义)
  - userId (谁创建的)

CheckInRecord
  - id, userId, categoryId
  - date, value (数值)
  - note, mood, photoUrl
  - timestamp

Challenge
  - id, creatorId, partnerId
  - title, categoryId, target, deadline
  - status (进行中/已完成/已失败)
  - loserId

CheerRecord
  - id, fromUserId, toUserId
  - type (戳一戳/加油/emoji)
  - timestamp
```
